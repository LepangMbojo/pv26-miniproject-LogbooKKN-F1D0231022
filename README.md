**Deskripsi Singkat**
Sistem Manajemen Tugas KKN adalah aplikasi desktop yang dirancang untuk mempermudah pencatatan, pembaruan, dan pemantauan daftar kegiatan harian selama program Kuliah Kerja Nyata (KKN). Aplikasi ini menyediakan antarmuka yang interaktif untuk mengelola data kegiatan secara lokal tanpa memerlukan koneksi internet.

**Teknologi yang Digunakan**
*   **Bahasa Pemrograman:** Python 3.x
*   **Framework GUI:** PySide6 (Qt for Python)
*   **Database:** SQLite (Bawaan Python)
*   **Styling Visual:** QSS (Qt Style Sheets)

**Cara Menjalankan**
1.  Pastikan Python 3 sudah terinstal di komputermu.
2.  Buka terminal atau command prompt, lalu instal *library* PySide6 dengan perintah:
    ```bash
    pip install PySide6
    ```
3.  Arahkan terminal ke dalam folder tempat file proyek ini disimpan.
4.  Jalankan aplikasi dengan mengetikkan perintah berikut:
    ```bash
    python main.py
    ```
